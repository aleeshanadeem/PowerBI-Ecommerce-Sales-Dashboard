# Ecommerce Sales Dashboard (Power BI)

![Uploading ecommerce-dashboard.png…]()


## Project Overview
This Power BI dashboard provides insights into ecommerce sales performance, customer demographics, product categories, regional sales, and monthly trends.

## Features
- Sales by Region Analysis
- Sales by Product Category
- Monthly Sales Trends
- Customer Age Analysis
- Quantity Distribution by Category
- Interactive Visualizations

## Tools Used
- Power BI
- Excel
- Data Visualization
- Data Analysis

## Key Insights
- Faisalabad generated the highest sales.
- Office Supplies contributed the largest category sales.
- October recorded the highest monthly sales.
- Customer age group 56-59 contributed the largest share of sales.
